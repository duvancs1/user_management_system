Website Documentation

Sign In/Create Account

  1. Visit the website's homepage.
  2. Click on the "Sign In" link or "Create Account" link.
  3. Fill in the required information for signing up or log in if you already have an account.
  4. Upon successful authentication, you will be redirected to the home page.

Navigation

  • The navbar on the left-hand side of the page contains links to:
    • Employee
    • Employee - Hierarchy
    • Technical Documentation

Home Page

  • On the home page, you will find quick links to the following features:
    • Add Employee
    • Employee - List
    • Employee - Hierarchy

Adding an Employee

  1. From the Home Page:
    • Click on the "Add Employee" button.
    • This action will open a modal popup prompting you to fill in the required fields for creating a user.
    • Here you can also set the reporting line manager, which would affect the employee hierarchy.
  2. From Employee List:
    • Navigate to the Employee List.
    • Click on the "Add User" button located at the top right corner of the page.
    • Fill in the necessary details in the modal popup that appears.

Admin User Creation

  • While adding a user, there is an option to create an admin user.
  • Check the designated tick box to designate the user as an admin.
  • Admin users have access to view all employee data, edit any employee information, and view salary details.

Editing User Information

  • For Regular Users:
    • Navigate to the Employee List.
    • Find the user whose information you want to edit.
    • Click on the "Edit" button located at the far right.
    • Update the user's details as needed.
  • For Admin Users:
    • Admin users have the privilege to edit all user information.
    • Follow the same steps as for regular users to edit any employee's information.

Deleting a User

  • Note: Only an admin user can delete users.
  • From the Employee List screen:
    • Locate the user you wish to delete.
    • Next to the "Edit" button, you'll find a "Delete" button.
    • Click on the "Delete" button to remove the user from the system.

Employee List

  • An avatar will display if you have a Gravatar account.

Employee Hierarchy

  • The top-level employee will be at the top of the hierarchy, with people who report to them listed underneath.
